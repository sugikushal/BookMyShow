# BookMyShow

## Requirements

### Actors

1. **Customer**
    1. Search based on
        - City
        - Location
        - Movie
        - Events
        - Theatres
    2. Book an event or movie
    3. Payment
    4. Cancel a booking
    5. View a booking
    6. Rate and review

2. **Admin**

    Add/Edit/Delete
    1. Theatre
    2. Screen and seats
    3. Movie
    4. Show

3. **Common**
    1. Registration
    2. Logging
    3. Customer Care

4. **System**
    1. Send email reagrding booking and status
    2. Send email on payment failure
    3. Initiate refund on cancellation
    4. Notification
